Ide Masih Ngambang
-----------------
Untuk ide game ini kurang tau susah atau bakal lumayan gampang..
niatnya bikin game kayak game2 jepang itu, gak tau tipe game apa namanya
biasa di sukai para wibu dan otaku

game ini beralur seperti game wibu pada umumnya, dimana user 
berusaha mendapatkan hati si perempuan yang telah ia pilih..

Karakter Game terdiri dari 3 orang cewek di kelas TID yaitu "Erna,Novi,Lala"
User bebas memilih cewek mana saja yang di inginkan untuk di jadikan tujuan PDKT
sehingga finishnya nanti sampe user berhasil membuat gadis tersebut menjadi pacarnya
